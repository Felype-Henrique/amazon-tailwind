# Como baixar o projeto 

```bash
 
 # Clonar repositório
 $ git clone https://github.com/Felype-Henrique/amazon-tailwind.git

 # Entrar no repositório
 $ cd amazon-tailwind

 # Executar
  Vá para src/index.html para ver a primeira visualização.
```
